The Google Doc used to host the content is located here.

To start working on your story, create a new Google Doc, get the id of that doc, replace the “id” in “config.json”. For this Google Doc, the id is the string on the URL between /d/ and /edit.

To learn more about coding with the template, click here.

I adapted it from The Pudding’s starter-template. Many thanks to Russell Goldenburg who developed this. I only changed the style and design to fit for NJAM. Apparently, there are lots of bugs need to be fixed.

I learned how to use it from The Pudding’s Youtube Channel. You don’t need to watch them all. I followed him for 20 episodes.
